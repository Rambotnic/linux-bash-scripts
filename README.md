## Keyboard Shortcuts
These are equivalent to Windows' shortcuts. `Meta` in this case would be the Windows logo key.

### Applications
* **Brave Browser**
	* Launch: `Meta + B`
* **KRunner**
	* Shortcut: `Meta + R`
* **System Monitor**
	* Shortcut: `Ctrl + Shift + Esc`
* **System Settings**
	* Shortcut: `Meta + I`

### System Services
* **Plasma**
	* Show Desktop: `Meta + M`
* **Session Management**
	* Lock Session: `Meta + L`

### Dolphin
* **Editable Location**
	* Shortcut: `F4`
* **Terminal**
	* Shortcut: `NONE`
* **Show Hidden Files**
	* Shortcut: `Alt + H, Alt + H`
